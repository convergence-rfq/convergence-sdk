Class TokenClient

This is a client for the Token module.

It enables us to interact with the SPL Token program in order to create, send, freeze, thaw, and mint tokens.

You may access this client via the tokens() method of your Convergence instance.

const tokenClient = convergence.tokens();
example

You can create a new mint account with an associated token account like so. The owner of this token account will, by default, be the current identity of the convergence instance.

const { token } = await convergence.tokens().createTokenWithMint();
